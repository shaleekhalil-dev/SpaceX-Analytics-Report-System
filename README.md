# SpaceX Falcon 9 Landing Success Analytics 🚀

A comprehensive data engineering and analysis system designed to predict and analyze the success factors of SpaceX Falcon 9 landings. This project integrates data scraping, processing, and automated reporting into a single, unified pipeline.

## 📊 Project Overview
This research focuses on the technical variables that influence the success or failure of rocket landings. By analyzing factors such as payload mass, orbit types, and launch sites, the system generates strategic insights to support operational excellence in aerospace engineering.

## ✨ Key Features
- **End-to-End Automation:** A "Maestro" script (`run_all.py`) orchestrates the entire workflow.
- **Data Synthesis & Analysis:** Processes complex launch data to identify success patterns.
- **Dynamic Visualizations:** Generates high-quality charts representing landing success rates and payload correlations.
- **Multi-Format Reporting:** Automatically exports findings into professional **PDF** and **Word (DOCX)** reports.

## 🛠️ Technical Stack
- **Language:** Python 3.x
- **Data Handling:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Document Generation:** FPDF2, Python-docx
- **Version Control:** Git & GitHub

## 📂 Project Structure
- `data/`: Contains the synthetic/scraped SpaceX datasets.
- `figures/`: Stores generated visual analytics (PNG/JPG).
- `outputs/`: The destination for finalized PDF and Word reports.
- `scripts/`: Modular Python scripts for data generation, processing, and visualization.
